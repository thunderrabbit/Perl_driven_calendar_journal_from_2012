Notes on how I set up Django to run a FB app on DH:

  1. set up domain to run as Passenger

  2. cd ~/domain.com

  3. wget http://wiki.dreamhost.com/django-setup.py

  4. python django-setup.py

  5. cd project_name

  6. git clone https://github.com/sciyoshi/pyfacebook.git
