Left home around noon thirty and got to the office at 1:45 to clean out my desk. But first I played online, trying to learn git a bit. I had downloaded lilURL from sourceforge, and made some updates. Wanted to upload those to github, but (fortunately) had the brainstorm that perhaps, just perhaps, someone had already uploaded it there. And someone had. I forked gbremer's copy of lilURL to thunderrabbit's copy of lilURL, and then did something like the following:

$ git clone git@github.com:thunderrabbit/lilurl.git

But I kept getting publickey denied errors (or something), and then I realized I needed to generate a public key on my server.

Did that, told github the public key part, and the clone worked perfectly.

$ git clone git@github.com:thunderrabbit/lilurl.git

Made my changes, (actually they were already made in a different folder, so I just copied them over) and then did

$ git add *

In the /trunk directory to add all the files.

Then

$ git commit

And that was it! (*)

(*) I think.
