I tweaked my sgml-mode.el code, (adding the end-of-line) so now if my point is still inside the </p> tag, I can do an html newline and it keeps the tags from getting confused.

(define-skeleton html-paragraph
  "HTML paragraph tag."
  nil
  (if (bolp) nil (progn (end-of-line) "\n\n"))
  "<p>" _ (if sgml-xml-mode ">/p>"))
