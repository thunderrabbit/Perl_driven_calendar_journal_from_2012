The text above is how much I wrote before I realized it would be way easier to record it first and have it automagically transcribed. I wondered about making a quick video using the built-in iSight. Found a reference to Quicktime Broadcaster which I downloaded only to find that while it can record to disk with a simple checkbox, it can't *not* broadcast just as easily. Someone said I can simply set the loopback to the local device, but didn't say how.

So then I found hackTV, but no compiled version. I'm all for open source and DIY, but I'm also for having someone else sorted out how to compile it so I don't have to!

Ah. Then I found http://developer.apple.com/samplecode/QuickTime/idxMovieCreation-date.html, and that it's got compiled versions.

Then I noticed a dmg link to WhackedTV, and it seems to work just about the way I want. No heaviness of iMovie, plus its time remaining clock counting UP while it exports.

WhackedTV produces a web appropriate file from the start. Or so it seems; the movie I made with iMovie was five minutes long, but the subsequent test with WhackedTV was only like 30 seconds.

Pausing while recording wih WhackedTV made the sound get outta wack.
