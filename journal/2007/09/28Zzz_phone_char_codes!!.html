The escape sequences above are for all the little icons that I can find that will display on my cell phone here in Japan. To see them, point your cell phone browser at http://keitai-codes.notlong.com. First, if necessary, come to Japan and get a cell phone.

The escape sequences consist of four parts:

  1. an escape character that looks like ^[
  2. $
  3. two digit code
  4. an escape character that looks like ^O

Using emacs, I can create the escape sequence for the first code via:

C-q C-[ $ E ! C-q C-o

I'm interested to know if it works on cell phones in other parts of the world.
